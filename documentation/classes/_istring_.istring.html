Static substr

  • substr(str: string, start: number, length?: number): string
    • Defined in IString.ts:368

    方法可在字符串中抽取从 start 下标开始的指定数目的字符。

    Parameters

    • str: string

      目标字符串

    • start: number

      要抽取的子串的起始下标。必须是数值。如果是负数，那么该参数声明从字符串的尾部开始算起的位置。也就是说，-1 指字符串中最后一个字符，-2 指倒数第二个字符，以此类推。

    • Default value length: number = str.length

      子串中的字符数。必须是数值。如果省略了该参数，那么返回从 stringObject 的开始位置到结尾的字串。

    Returns string

    一个新的字符串，包含从 stringObject 的 start（包括 start 所指的字符） 处开始的 length 个字符。如果没有指定 length，那么返回的字符串包含从 start 到 stringObject 的结尾的字符。
